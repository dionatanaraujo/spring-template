# Estrutura de folders

Um example de estrutura de folders para trabalhar com REST API.

```bash
├── src/main/java
│   ├── config
│   ├── exception
│   ├── converter
│   ├── integration
│   │   ├── client
│   │   ├── model
│   │   │   ├── dto
│   ├── model
│   │   ├── dto
│   │   ├── entity
│   │   ├── enumeration
│   ├── repository
│   ├── resource
│   ├── service
│   ├── util
├── src/main/resources
│   ├── application.yml
│   ├── messages
├── src/test/java
├── README.md
└── .gitignore
└── .editorconfig
```
